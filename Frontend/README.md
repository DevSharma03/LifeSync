## Getting Started

### Prerequisites

- Node.js (recommend v14+)
- npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/DevSharma03/LifeSync.git
    ```
2. Navigate to the frontend directory:
    ```bash
    cd LifeSync/Frontend
    ```
3. Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the Application

```bash
npm start
# or
yarn start
```
