## вңЁ What is LifeSync?

**LifeSync** is a production-grade, full-stack personal assistant platform that uses machine learning and AI to provide intelligent insights across four life dimensions:

- **рҹҸҘ Health** вҖ“ Sleep tracking, exercise monitoring, nutrition analysis
- **рҹ§  Mental Wellness** вҖ“ Stress management, mood tracking, mindfulness support
- **вҡЎ Productivity** вҖ“ Task optimization, focus improvement, time management
- **рҹ’° Financial Health** вҖ“ Expense tracking, savings goals, financial wellness

## рҹҺҜ Core Features

### рҹӨ– AI & Machine Learning
- **Cold-Start Personalization** вҖ“ Intelligent user profiling from minimal initial data
- **Gaussian Mixture Models** вҖ“ 12-cluster user segmentation for personalized recommendations
- **LightGBM Cascade Prediction** вҖ“ 5-tier predictive models (Health вҶ’ Mind вҶ’ Productivity вҶ’ Finance вҶ’ Life Score)
- **Groq AI Integration** вҖ“ Fast, low-latency LLM responses for conversational AI
- **Feature Influence Mapping** вҖ“ Explainable AI showing how features impact life scores

### рҹ’ҫ Data Architecture
- **Multi-Dimensional Features** вҖ“ 46+ tracked metrics across 4 life dimensions
- **Semantic Constraints** вҖ“ Intelligent validation firewall for data integrity
- **Normalized Entropy Tracking** вҖ“ Confidence scoring for predictions
- **Production-Grade Models** вҖ“ Trained on 20,000+ synthetic user profiles

### рҹӣ пёҸ Technical Stack
- **Backend** вҖ“ Node.js + Express REST API
- **Frontend** вҖ“ React with responsive UI
- **AI/ML** вҖ“ Python FastAPI microservice with scikit-learn, LightGBM
- **Models** вҖ“ Clustering, prediction, and profiling pipelines
- **Deployment** вҖ“ Docker-ready, containerized architecture

## рҹ“ҡ ML Models Overview

### рҹ§© Cold-Start Personalization

**Input:** 15 initial user assessment scores (0-100 scale)
```python
[
  "sleep_quality", "physical_activity_consistency", "diet_quality",
  "daily_energy_level", "sedentary_level", "stress_level", "anxiety_level",
  "mood_stability", "mindfulness_habit", "social_support", "focus_ability",
  "task_completion_reliability", "distraction_level", "financial_discipline",
  "financial_stress"
]
```

**Output:** Complete user profile with 46 features

**Algorithm:** Gaussian Mixture Model (12 clusters) + Feature Influence Mapping
- **Clustering Accuracy:** 40.6% top-1, 83% top-3
- **Profile Generation Error:** 27.5% normalized MAE
- **Confidence Scoring:** Normalized entropy-based confidence

---

### рҹҺҜ Cascade Prediction Model

**5-Tier LightGBM Cascade:**

```
User Features (46 inputs)
        вҶ“
    Health Model  (10 features) вҶ’ Health Score
        вҶ“
    Mind Model    (13 features + Health) вҶ’ Mind Score
        вҶ“
    Productivity  (10 features + Health + Mind) вҶ’ Productivity Score
        вҶ“
    Finance Model (8 features + Productivity) вҶ’ Finance Score
        вҶ“
    Life Model    (4 pillars) вҶ’ Overall Life Score
```

**Features Per Model:**
- **Health:** Sleep, exercise, diet, hydration, steps, calories, sedentary time
- **Mind:** Depression, anxiety, stress, trauma, mood, meditation, breathing, support
- **Productivity:** Tasks, completion, efficiency, focus, energy levels
- **Finance:** Budget, expenses, savings, emergency fund, debt
- **Life:** Weighted combination of all four pillars
